ตรวจสอบการตั้งค่า Supabase

หน้านี้ช่วยตรวจว่า config.js และฐานข้อมูลพร้อมใช้งานหรือยัง

ขั้นตอนที่ต้องทำใน Supabase Dashboard

  1. SQL Editor → วางและรัน supabase-schema.sql ทั้งไฟล์
  2. Database → Replication → เปิด Realtime สำหรับ messages, queue, rooms
  3. Authentication → Sign In / Providers → เปิด Anonymous Sign-Ins
  4. Authentication → Users → สร้างบัญชีแอดมิน (อีเมล/รหัสผ่าน)
  5. SQL Editor → รัน: UPDATE profiles SET status = 'admin' WHERE id = 'uuid-ของแอดมิน';
